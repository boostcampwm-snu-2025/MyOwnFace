# 📌 MyOwnFace Frontend — Recommended Libraries Setup

MyOwnFace 앱의 핵심 기능(얼굴 분석 · 맞춤형 메이크업 튜토리얼 · 제품 추천)을 구현하기 위해 필요한 **React + Vite 프론트엔드 라이브러리 목록**입니다.

---

## 🔍 1. 얼굴 분석(Feature Extraction)

| 목적             | 라이브러리             | 설명                                        |
| ---------------- | ---------------------- | ------------------------------------------- |
| 얼굴 특징점 추출 | `@mediapipe/face_mesh` | 눈·코·입·얼굴형 등 468개 얼굴 랜드마크 분석 |
| 보조             | `face-api.js`          | 기본 얼굴 탐지 및 표정 분석                 |

---

## 📁 2. 이미지 업로드 & 전처리

| 목적           | 라이브러리                  | 설명                                |
| -------------- | --------------------------- | ----------------------------------- |
| 이미지 업로드  | `react-dropzone`            | Drag & Drop 방식 이미지 선택        |
| EXIF 정보 처리 | `exifr`                     | 이미지 회전 정보 및 메타데이터 파싱 |
| 이미지 압축    | `browser-image-compression` | 업로드 전 이미지 용량 최적화        |

---

## 🧠 3. 상태관리 & API 통신

| 목적           | 라이브러리              | 설명                              |
| -------------- | ----------------------- | --------------------------------- |
| 상태관리       | `zustand`               | 분석 결과/옵션/튜토리얼 상태 저장 |
| 서버 통신      | `axios`                 | API 호출                          |
| 서버 상태 관리 | `@tanstack/react-query` | 캐싱, 로딩, 에러 핸들링 최적화    |

---

## 🎨 4. UI & 얼굴 랜드마크 오버레이

| 목적               | 라이브러리       | 설명                         |
| ------------------ | ---------------- | ---------------------------- |
| 스타일링           | `tailwindcss v4` | UI 커스터마이징              |
| UI 컴포넌트        | `shadcn/ui`      | 모던 UI 기반 컴포넌트        |
| Canvas 기반 시각화 | `react-konva`    | 얼굴 랜드마크 점/라인 그리기 |

---
